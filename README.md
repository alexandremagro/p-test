# P-test

### Prerequisites

- Ruby 2.7.0
- PostgreSQL

### Installation

Install dependencies:

```
bundle && yarn
```

Initialize local databases:

```
rails db:create db:migrate
```

## Development

Running locally:

```
rails s
```

Running tests:

```
rails test
rails test:system
```
